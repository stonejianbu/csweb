# csweb
Being the cornerstone framework of the web for golang

## Characteristic

1. provides HTTP and GRPC serve
2. validator
3. metrics (prometheus)
4. tracer (zipkin)
5. log with traceID
6. tools library (gorm/jwt/seata...)
7. jwt Auth
8. recovery
9. ratelimit
10. error handle
